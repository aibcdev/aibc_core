# Adding and Using Images in AIBC

Images are a crucial part of content. AIBC supports adding images to your content in multiple ways: uploading your own, using images from your Media Library, or generating AI images.

## Ways to Add Images

### 1. Upload Your Own Images

**When creating content:**

1. Go to **Content Hub** → Create or edit content
2. In the content editor, find **"Add Media"** or **"Images"** section
3. Click **"Upload Image"**
4. Select image(s) from your device
5. **Supported formats:**
   - JPG/JPEG
   - PNG
   - GIF
   - WebP
6. Images are uploaded and attached to content

**From Media Library:**

1. In content editor, click **"Choose from Library"**
2. Browse your **Brand Assets** → **Media** library
3. Select image(s)
4. Click **"Add to Content"**

### 2. Use Images from Media Library

Your Media Library (in **Brand Assets** → **Media** tab) stores all your brand images.

**To use library images:**

1. When creating/editing content, click **"Media Library"**
2. Browse or search for images
3. Select image(s)
4. Click **"Add"**

**Benefits:**
- Reuse brand assets
- Consistent visual identity
- Easy access to approved images
- Organized by category

### 3. Generate AI Images (Production Room)

AIBC can generate images using AI in the Production Room.

**To generate AI images:**

1. Go to **Production Room**
2. Select **"Image"** as output type
3. **Configure:**
   - **Duration/Style**: Short, Mid, Long (affects complexity)
   - **Visual Style**: Animation, Corporate, Minimalist, etc.
   - **Aspect Ratio**: 1:1, 16:9, 9:16, etc.
   - **Description**: What you want in the image
4. **Check credit cost** (varies by style and complexity)
5. Click **"Generate"**
6. Image is added to production queue
7. When complete, use in content

**Credit costs:**
- **Short/Simple**: Lower credit cost
- **Mid/Moderate**: Medium credit cost
- **Long/Complex**: Higher credit cost
- Check costs in Production Room before generating

## Image Requirements by Platform

Different platforms have different image requirements:

### Instagram

**Feed Posts:**
- **Aspect ratio**: 1:1 (square) or 4:5 (portrait)
- **Size**: 1080x1080px (square) or 1080x1350px (portrait)
- **Format**: JPG or PNG
- **Max file size**: 8MB

**Stories:**
- **Aspect ratio**: 9:16 (vertical)
- **Size**: 1080x1920px
- **Format**: JPG or PNG
- **Max file size**: 8MB

**Reels:**
- **Aspect ratio**: 9:16 (vertical)
- **Size**: 1080x1920px
- **Format**: MP4 (video) or JPG/PNG (cover)
- **Max file size**: 100MB (video)

### Twitter/X

**Images:**
- **Aspect ratio**: 16:9 (landscape) recommended
- **Size**: 1200x675px (optimal)
- **Format**: JPG or PNG
- **Max file size**: 5MB

**Multiple images:**
- Up to 4 images per tweet
- Can mix aspect ratios
- All images shown in grid

### LinkedIn

**Posts:**
- **Aspect ratio**: 1.91:1 (landscape) recommended
- **Size**: 1200x627px (optimal)
- **Format**: JPG or PNG
- **Max file size**: 5MB

**Articles:**
- **Cover image**: 1200x627px
- **Inline images**: Flexible, but 1200px width recommended

### Facebook

**Posts:**
- **Aspect ratio**: 1.91:1 (landscape) or 1:1 (square)
- **Size**: 1200x630px (landscape) or 1200x1200px (square)
- **Format**: JPG or PNG
- **Max file size**: 8MB

### YouTube

**Thumbnails:**
- **Aspect ratio**: 16:9
- **Size**: 1280x720px (optimal)
- **Format**: JPG or PNG
- **Max file size**: 2MB

### TikTok

**Images:**
- **Aspect ratio**: 9:16 (vertical)
- **Size**: 1080x1920px
- **Format**: JPG or PNG
- **Max file size**: 10MB

## Replacing Images on Content

**To replace an image on existing content:**

1. Go to **Content Hub**
2. Find the content with the image
3. Click to open content details
4. Find the image section
5. Click **"Replace Image"** or image icon
6. **Choose replacement method:**
   - Upload new image
   - Select from Media Library
   - Generate new AI image
7. New image replaces old one
8. Save content

**Bulk replace:**
- Select multiple content items
- Use bulk action **"Replace Images"**
- Choose replacement source
- Apply to all selected

## Image Optimization

### Before Uploading

**Best practices:**
- **Compress images**: Reduce file size without losing quality
- **Optimize format**: Use JPG for photos, PNG for graphics
- **Correct dimensions**: Match platform requirements
- **Check file size**: Stay under platform limits

### In AIBC

**Automatic optimization:**
- AIBC may compress images automatically
- Formats are converted if needed
- Dimensions are adjusted for platform
- Quality is maintained

**Manual optimization:**
- Use image editing tools before uploading
- Compress with tools like TinyPNG
- Resize to platform requirements
- Convert formats if needed

## Organizing Images

### In Media Library

**Organize by:**
- **Category**: Logos, Products, Team, etc.
- **Campaign**: Group by campaign or project
- **Platform**: Organize by platform requirements
- **Date**: Chronological organization

**Add metadata:**
- **Tags**: Add tags for easy searching
- **Descriptions**: Describe image content
- **Alt text**: For accessibility (coming soon)

### In Content

**Image management:**
- Images are attached to specific content
- Can be reused across multiple content items
- Replace without affecting other content
- Delete from content without deleting from library

## Using AI-Generated Images

### When to Use AI Images

- **No suitable image available**: Generate custom image
- **Specific concept**: Need image for specific idea
- **Brand consistency**: Generate images matching brand style
- **Cost-effective**: Cheaper than stock photos (with credits)

### Generating AI Images

1. Go to **Production Room**
2. Select **"Image"** output type
3. **Describe what you want:**
   - Subject matter
   - Style (matches your brand)
   - Colors
   - Composition
4. **Select style and aspect ratio**
5. **Check credit cost**
6. Generate and wait for completion
7. Use in content

### Best Practices for AI Images

- **Be specific**: Detailed descriptions get better results
- **Match brand style**: Use your brand colors and style
- **Consider platform**: Generate for specific platform requirements
- **Review before using**: Check quality and relevance
- **Edit if needed**: Use image editing tools for final touches

## Image Credits and Costs

**Credit costs for AI images:**
- Varies by complexity and style
- Check costs in Production Room
- Costs deducted from credit balance
- Free tier has limited image generation

**Managing costs:**
- Use Media Library when possible
- Upload your own images (no credit cost)
- Generate only when needed
- Choose simpler styles for lower costs

## Accessibility

### Alt Text

**Adding alt text:**
- Describe image content
- Helps with accessibility
- Improves SEO
- Required for some platforms

**Best practices:**
- Be descriptive but concise
- Include key information
- Don't start with "Image of..."
- Focus on content and context

### Image Descriptions

**In content:**
- Add captions when relevant
- Describe images in post text
- Use for context and engagement

## Common Issues

### Image Too Large

**Solution:**
- Compress before uploading
- Use image optimization tools
- AIBC may compress automatically
- Check platform file size limits

### Wrong Aspect Ratio

**Solution:**
- Resize to platform requirements
- Use image editing tools
- Generate AI images with correct aspect ratio
- AIBC may adjust automatically

### Image Not Uploading

**Solution:**
- Check file format (JPG, PNG, GIF, WebP)
- Verify file size under limit
- Check internet connection
- Try different browser
- Contact support if persists

### Poor Image Quality

**Solution:**
- Use higher resolution source images
- Don't over-compress
- Match platform requirements
- Use appropriate format (JPG for photos, PNG for graphics)
