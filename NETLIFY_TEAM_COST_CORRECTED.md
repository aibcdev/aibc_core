## How to Add Team Members (Pro Plan Only)

### Step 1: Upgrade to Pro
1. Go to: https://app.netlify.com
2. Click **"Upgrade"** or go to billing
3. Select **Pro plan**
4. You'll be billed $20/month for yourself

### Step 2: Add Team Members
1. Go to **Team settings** (now visible after upgrade)
2. Click **"Invite members"**
3. Enter email addresses
4. Each additional member = **+$20/month**

## What This Means for You

### If You Need Team Members:

**Option 1: Upgrade to Pro**
- $20/month base + $20/month per team member
- Full team collaboration
- All Pro features

**Option 2: Share Account**
- One person manages Netlify
- Others can work via GitHub
- Deployments happen automatically
- No extra cost, but limited access

**Option 3: Use GitHub for Collaboration**
- Team members work in GitHub
- Netlify auto-deploys from GitHub
- Everyone can push code
- Only you manage Netlify account

## Recommendation

### For Small Teams:
- **Use GitHub for collaboration** (free)
- **One person manages Netlify** (free tier)
- **Auto-deploy from GitHub** (works on free tier)
- Everyone can contribute code
- Only Netlify account owner manages deployments

### For Larger Teams:
- **Upgrade to Pro** if you need:
  - Multiple people managing Netlify
  - Different access levels
  - Team management features
  - $20/month per person
