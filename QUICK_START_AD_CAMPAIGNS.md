## üîß Optimization Tips

### Week 1-2: Let It Run
- Don't make changes too quickly
- Let Google Ads learn
- Collect at least 100 clicks before optimizing

### Week 3-4: Optimize
- Pause ads with CTR < 2%
- Increase bids on ads with CTR > 5%
- A/B test headlines
- Refine negative keywords

### Month 2+: Scale
- Double budget on winning campaigns
- Expand to similar keywords
- Create ad variations for winners
- Test new landing pages

## üö® Common Mistakes to Avoid

1. **Don't start with high-competition keywords** - Start with zero-competition keywords first
2. **Don't change bids too frequently** - Let Google Ads learn (wait 3-7 days)
3. **Don't ignore negative keywords** - Add them to avoid irrelevant traffic
4. **Don't use generic landing pages** - Create specific pages for each ad group
5. **Don't forget conversion tracking** - Set it up before launching
6. **Don't set budgets too low** - Need at least $30-50/day for meaningful data
7. **Don't pause ads too quickly** - Wait for at least 100 clicks before judging
